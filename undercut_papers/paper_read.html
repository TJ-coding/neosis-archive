Cosine Similarity as Logits? A Scalable Knowledge Probe Using Embedding Vectors from Generative Language Models

Jinno, Hayashi, Sakai, Kamigaito, Watanabe · EACL 2026 · 2026.eacl-long.382 · pp. 8188–8200
  • O(n²) → O(n) scaling for knowledge probes
  • Probes relational knowledge with decoder embeddings from generative PLMs
  • Supports rank-based metrics (Hit@k), not just accuracy
  • Handles multi-token entities and disambiguates homographic names

Abstract

Existing probes for generative PLMs scale quadratically O(n²) with the number of entities, limiting usable knowledge-graph size.

DEER uses decoder embeddings for linear O(n) probing with rank-based metrics, multi-token support, and automatic disambiguation of homographic names. Empirically correlates with existing probes.
